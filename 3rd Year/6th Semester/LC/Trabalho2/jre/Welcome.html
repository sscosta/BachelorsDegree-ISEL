Welcome to Java™ 2

Welcome to the Java™ 2 Standard Edition Runtime Environment. This provides complete runtime support for Java 2 applications.

The runtime environment includes the Java Plug-in which supports running the Java 2 environment inside Internet Explorer or Netscape Navigator.

Note that the Java 2 runtime environment does not replace the existing Java environment provided with your browser. Old applets will still get the browser's default Java environment. New web pages designed for Java 2 can use special HTML tags to invoke the Java 2 Runtime Environment.

Java Plug-in Control Panel

The Java Plug-in control panel is now in the Windows control panel folder. Sophisticated users can use this control panel to adjust settings for the Java Plug-in.
